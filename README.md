# Boilerplate

Personal boilerplate, used for client projects.

Consists of :
- React-Redux
- Jspm (bundler)
- Express (server)
- Handlebars (server-side templating)
- Mongodb

## Setup

    cd src
    yarn --no-bin-links
    jspm install -y

# Run

    cd src
    node index

# Build

    sh ./build.sh
